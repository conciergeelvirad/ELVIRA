## üìù Maintenance Guidelines

### Adding New Documentation

1. Check if it fits in existing docs first
2. If new doc needed, add to root level
3. Update README.md documentation index
4. Keep focused and up-to-date

### Updating Documentation

1. Update the relevant active document
2. Update last modified date
3. If superseding old doc, move old to archive
4. Update archive README if needed

### Archiving Documentation

1. Move completed/superseded docs to `docs/archive/`
2. Update `docs/archive/README.md` with entry
3. Update root README.md to reflect change
4. Commit with clear explanation
